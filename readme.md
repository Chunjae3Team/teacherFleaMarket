# 서비스 구성
| 기능 분류 | 프로그램 내 구현 형태 |
| --- | --- |
| CREATE | 회원가입, 새 즐겨찾기 목록 생성, 영수증 생성 |
| READ | 즐겨찾기 목록 불러오기, 구매 내역 불러오기 |
| UPDATE | 즐겨찾기 목록 추가, 구매 내역 추가, 회원 등급 업데이트, 배달 현황 업데이트 |
| DELETE | 구매 내역 삭제 |
